Network Engineer

Saya memiliki keahlian di bidang jaringan dan telah berpengalaman dalam konfigurasi serta pengaturan perangkat jaringan seperti router WiFi, Mikrotik, dan Cisco. Saya juga menangani jaringan parabola dan memiliki kemampuan dalam mengelola sistem komunikasi menggunakan Trixbox. Selama 4 bulan praktik kerja lapangan, saya mempelajari secara mendalam berbagai aspek jaringan, termasuk penggunaan MikroTik OS dengan pendekatan konfigurasi berbasis Command Line Interface (CLI).

Selain itu, saya telah menggunakan aplikasi Cisco untuk merancang model topologi jaringan sebelum instalasi di lapangan. Dengan keterampilan ini, saya mampu menangani kebutuhan jaringan dari skala kecil hingga menengah dengan efisien dan andal.
